
 Журнал изменений в WWWTool
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

[+] Новая возможность
[-] Исправленная ошибка
[*] Изменение

Версия 0.5 (от 17 февраля 2011)
  [*] Добавлена обязательная контрольная сумма к штрих-коду, 
      несовместимая с предыдущими версиями программы

Версия 0.4 (от 11 февраля 2007)
  [+] Возможно переименование команд, смена столика
  [+] Стоимость ответа может варьироваться от команды к команде
  [+] Команды могут не давать ответа (спортивная Своя игра)

Версия 0.3 (от 3 января 2007)
  [+] У каждого вопроса может быть собственная стоимость

Версия 0.3 (от 17 декабря 2006)
  [+] Использование нескольких шаблонных файлов одновременно

  [*] Мышка теперь не так опасна: при включенном режиме сканера
      щелчки не влияют на правильность ответов
  [*] Упаковка UPXом

Версия 0.3 (от 19 января 2006)
  [+] Печать бланков ответов команд

Версия 0.3 (от 4 января 2006)
  [+] Работа со сканером

  [-] При продолжении игры из файла default.state он читался два раза

  [*] Компактный вид результатов игры

Версия 0.2 (зимняя, от 1 декабря 2005)
  [+] Файлы *.state открываются из Проводника Windows
  [+] Команды, полностью решившие раунд, подсвечиваются

  [-] Все время выкидывало на "Высшую лигу" в обычной игре
  [-] Неправильный вызов функции format при добавлении команды

Версия 0.2 (от 7 ноября 2005)
  [-] Не менялось состояние зачет/незачет
  [-] Не создавалось расширение по умолчанию для сохраняемых файлов (html, state)

Версия 0.2 (от 18 июня 2005)
  [+] Документация соответствует возможностям программы

  [*] Все сохранение переехало в один пункт меню

Версия 0.2 (от 14 февраля 2005)
  [+] Загрузка состояния по умолчанию - файл ./default.state
  [+] Аннулирование вопроса
  [+] Пометка после перехода

  [*] В диалоговых окнах, где нужно указать число, вместо TEdit используется TSpinEdit
  [*] Контроль за номером столика при переходе к команде
  [*] Весь интерфейс (кроме шапки GNU GPL) теперь на русском (никаких OK-Cancel)
  [*] Сохраняется теперь и состояние таймера
  [*] Исходные тексты программы переехали в каталог ./src/

  [-] Для всех TSpinEdit написан обработчик нажатий на клавиатуру
      (ранее в модальных диалоговых окнах не было реакции на Enter и Escape)
  [-] При переходе к команде за столиком по умолчанию показывается текущая команда
      (ранее было количество строк в таблице)

Версия 0.1 (от 8 февраля 2005)
  Первая официальная версия.
